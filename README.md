# Gemini Admin

**Gemini Admin** 是一个通用后台管理系统（Admin Panel），基于 **React** 构建，面向创业团队和中小型项目。当前版本实现了最基础的 **MCP（Minimal Control Panel）** 功能，后续将逐步演化为完整的企业级后台系统。

---

## 🎯 目标

- 提供通用后台管理框架，快速构建项目管理系统
- 支持最小化控制面板（MCP）功能，便于快速迭代
- 后续演化支持更多高级功能，包括权限管理、数据可视化、复杂报表、插件化扩展等

---

## ⚡ 当前特性（MCP 简化版）

- **基础 CRUD**：动态表格、增删改查操作
- **用户管理**：用户列表、简单角色配置
- **数据展示**：基础统计图表
- **菜单管理**：可快速配置的侧边栏菜单
- **轻量可扩展**：React 前端架构便于后续模块扩展
- **Docker 支持**：快速部署和启动

---

## 🛠 技术栈

| 层级       | 技术                     |
|------------|-------------------------|
| 前端       | React + Vite            |
| UI         | Ant Design / Tailwind UI |
| 状态管理   |  Zustand         |
| 路由       | React Router            |
| 后端       | Spring Boot |
| 数据库     | MySQL      |
| 容器化     | Docker                  |
